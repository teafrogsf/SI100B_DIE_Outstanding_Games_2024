## 游戏操作
A D 移动

SPACE 跳跃

T 跟附近的NPC对话

F 发射魔法（这一能力会在获得神奇妙妙工具后生效）

Q 退出任何界面

你可以在任何游戏关卡内按ESC进入暂停菜单，在任何菜单（开始/暂停/死亡/胜利）都有相关操作的指示，此处略
